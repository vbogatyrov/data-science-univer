To find similar lets calculate Euclidian distance between regions, and find pairs of regions with a close distance between each other.

Basing on the histogram, there are some eruptions of distance, which can be dropped

Lets have a closer look at distance distribution density:

To find the smallest distances, lets take the following approach:

min_distances <- region_dist_cleaned[region_dist_cleaned < (median(region_dist_cleaned) - 1.09*sd(region_dist_cleaned))]
## [1] 5069770 6972834 6313226 6729214 7009040

For ease of finding pairs of regions with the given distance between them, lets convert distances vector to a matrix, in which element (i,j) represents distance between i and j regions.

region_dist_matr <- as.matrix(region_dist) # Create full matrix from dist. object

To finds indexes of elements with the given values in the matrix lets use the following custom function:

indexOf <- function(matr, vals) {
    res <- c()
    row_num <- dim(matr)[1]
    col_num <- dim(matr)[2]
    for (val in vals) {
        for (i in 1:row_num) {
            for (j in 1:col_num) {
                if (abs(val - matr[i,j]) < 1) {
                  res <- c(res, c(val, i,j))
                  break
                }
            }
        }
    }
  return(res)
}
